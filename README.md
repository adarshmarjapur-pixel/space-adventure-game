# space-adventure-game
A 3D space adventure shooting game built with Unity
Assets/Scripts/PlayerController.cs - Player movement and shooting controls
Assets/Scripts/EnemyAI.cs - Enemy AI behavior and movement patterns
Assets/Scripts/GameManager.cs - Game state, enemy spawning, and score management
Assets/Scripts/Bullet.cs - Bullet physics and collision handling
Assets/Scripts/UIManager.cs - HUD, score display, and UI management
Assets/Scripts/CameraController.cs - Smooth camera following mechanics
